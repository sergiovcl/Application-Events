certo, estes conceitos são bem simples:
pense da seguinte forma existem três partes de uma página web:
   * HTML, a estrutura, ou seja, é como você organiza os elementos na página
   * CSS, o estilo, é como você define a aparência dos elementos na página
   * JS, é o que dá vida aos elementos da página, através do JS você consegue acessar atributos da estrutura, de um elemento, ou seja o HTML
     e também do estilo do elemento, o CSS

daí chegamos as suas dúvidas:
   * id e class são atributos HTML e tem os seguintes propósitos:
       - identificar um elemento, pra que você possa alterar suas propriedades no JS e no CSS
     eles são o famosos "seletores", justamento por que permitem que você selecione um elemento em um documento HTML

     É muito importante que você saiba o seguinte:
       - classes, são seletores de N elementos com o mesmo nome de classe
       - Id é um seletor para um único elemento no documento.

   * var é referente somente ao javascript, ela é usada para definir uma variável local, vamos tratar disso mais adiante.
